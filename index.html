Mind Bound

Easy-to-use and multi-user calendar application made using Java

The Mind Bound is my first project working with OOP, various data structures, file structures, graphics, etc. The goal of this project was to design and launch a calendar application with a multi-user function, as well as a to-do-list function

The program takes in, and remembers different users and store their data in a binary file. Each users are able to create events by a to-do-list addon on the calendar. This was approached via LinkedLists and Inheritance.

The calendar itself was designed using hitboxes of 2D arrays, and the calendar data, such as Years, Months, were approached via Inheritance, LinkedLists, and manipulation of text files.

Fuel Cell Car

Hydrogen Fuel-celled Toy Car

The goal of this project was to build a car powered by hydrogen fuel-cells. The additional factor to this project was to create a line-following algorithm, so that the fastest team that completed the line-mze tracks got the highest mark.

This project was a mix of three core courses, CHE102 (Chemistry), MTE100 (Mechatronics Engineering), GENE121 (C++). Our team had to research the pressure vessels associated with each fuel-cell, as well as the gas transfer to maximize the run-time. Once the research was done, we performed various chemical calculations to determine how much hydrogen fuel was created. As for the programming, we were given a designated IDE to successfully code the line-following algorithm. Bascially, the algorithm would allow the robot to follow the line-maze around the tracks and get to the endpoints without interffering with the obstacles.

This was my very first group project with my cohorts, and it was really interesting how I was able to integrate core courses to build a robot that runs on hydrogen-cells and navigate itself on line-maze.
